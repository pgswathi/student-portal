This is a student web portal where students marks,attendance can be monitored by staff. 
Placement details, games, chatbot is additionally available for students.
There is 3 interface for student, staff and admin.

Student page can be launched by executing "launch.html" page in browser.
Staff page can be launched by executing "slaunch.html" page in browser.

Install WAMP/LAMP server.
Install phpmyadmin and import "portal.sql" file into it for database tables.

FRONTEND: HTML, CSS, JS, BootStrap.
BACKEND: PHP, MySql.
